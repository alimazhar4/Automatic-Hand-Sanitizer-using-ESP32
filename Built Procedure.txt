To build this project, you will require the following equipment:
- Bread Board
- ESP 32 Microcontroller
- Male to male wires 
- Male to Female wires
- Micro Servo Motor (SG90 was used in this project)
- Ultrasonic sensor (HC-SR04 was used in this project)

Install the ESP32 setup in your arduino IDE if not already setup. Moreover, also you might need to install the ESP32 Servo library separately. The zip file of that library used is already given in this repository by the name "ServoESP32-master.zip"
